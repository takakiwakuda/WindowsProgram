# Get-WindowsProgram

## SYNOPSIS

Retrieves installed programs on Windows.

## SYNTAX

```powershell
Get-WindowsProgram [<CommonParameters>]
```

## DESCRIPTION

Retrieves installed programs on Windows.

## EXAMPLES

### Example 1

```powershell
PS C:\> Get-WindowsProgram
```

Retrieves installed programs on your Windows.
